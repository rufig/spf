ОПИСАНИЕ
--------

SP-FORTH это надёжная и удобная форт система генерирующая оптимизированный
машинный(native) код для процессоров Intel x86. Работает на Windows 9x, NT
(существуют также порты на Linux, Kolibri OS).

SP-Forth является свободным программным обеспечением. Вы вправе распространять
и/или модифицировать его в соответствии с условиями Стандартной Общественной
Лицензии GNU, опубликованной Free Software Foundation.  Для получения более
подробной информации ознакомьтесь с [docs/license/gpl.ru.txt](license/gpl.ru.txt)

Мы распространяем данную программу в надежде на то, что она будет вам полезной,
однако НЕ ПРЕДОСТАВЛЯЕМ НА НЕЕ НИКАКИХ ГАРАНТИЙ, в том числе ГАРАНТИИ ТОВАРНОГО
СОСТОЯНИЯ ПРИ ПРОДАЖЕ и ПРИГОДНОСТИ ДЛЯ ИСПОЛЬЗОВАНИЯ В КОНКРЕТНЫХ ЦЕЛЯХ.


УСТАНОВКА
---------

SP-Forth для Windows распространяется как RAR архив или как
самоустанавливающийся исполняемый файл.

Если вы устанавливаете из архива - распакуйте его в желаемую папку. Всё,
теперь он готов к использованию. Если же вы хотите ассоциировать `*.f` и `*.spf`
файлы с `spf4.exe` - запустите `docs/install/install.bat`, который покажет GUI
окошко для настройки реестра. Или вы можете вручную отредактировать файл
`docs/install/spf_path_install.reg` (укажите правильный путь к `spf4.exe`) и
запустите его. Теперь вы можете писать свой код, сохранить его в `*.f` файл и
запустить на исполнение двойным кликом мышью.

Если у вас обычный исполняемый инсталлятор - просто запустите его. Мастер
настройки проведёт вас через установочный процесс.

Смотрите файл [docs/whatsnew.ru.txt](whatsnew.ru.txt) с кратким описанием недавних изменений.

Каталоги:

* `/devel`   - дополнительные библиотеки и примеры
* `/docs`    - документация
* `/lib`     - стандартные библиотеки, обычные ANS и не-ANS расширения такие как `float.f`, `locals.f` ...
* `/samples` - отлаженные GUI и консольные примеры.
* `/src`     - полные исходники с комментариями и `compile.bat` скриптом

Файлы в корневом каталоге:

* `help.fhlp`        - подключается по умолчанию расширением `lib/ext/help.f`
* `jpf375c.exe`      - более старая версия СПФ для пересборки
* `spf4.exe`         - SPF ;)
* `spf4.ini`         - этот файл автоматически подключается SPF'ом при старте
